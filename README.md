Transribe you Audio!!

This is a barebones readme just to get you started.

I am using UV instead of PIP.

1. Clone the repo
2. Run `Uv Sync`
3. Set up your required config in config.py for first launch
3. Run `Uv run app.py`
4. Wait for the model to download during first launch depending on the whisper model chosen
5. Enjoy transcribing!!
